# Point of Sale System

Inventory management system for small businesses with sales tracking and receipt generation.

## Features (In Development)
- Product inventory management
- Sales transactions
- Receipt generation
- Product categories/tags
- Sales history

## Tech Stack
- **Backend:** Node.js, Express
- **Database:** MySQL
- **ORM:** Sequelize

## Database Schema
- Products
- Sales Tickets
- Product Tags/Categories
- Sales-Products (junction table)

## Status
🚧 Work in progress - Currently developing the backend and database structure.

## Installation
```bash
# Clone repository
git clone [your-repo-url]

# Install dependencies
npm install

# Configure database
# Edit config files with your MySQL credentials

# Run
npm start
```
